## 3. WebSDK 嵌入模块

### 3.1 知识库 WebSDK

**操作流程**:
1. 用户点击"制度知识库"卡片的"启动"按钮
2. 调用 `POST /api/resources/kb-policy/launch`
3. 获取 `launch_id` 后导航到 `/launch/{launchId}`
4. WorkspacePane 组件加载
5. 调用 `GET /api/launches/{id}/embed-config` 获取配置
6. 渲染 iframe 加载 `/sdk-host.html`
7. iframe 通过 postMessage 接收配置，加载 WebSDK

**预期浏览器显示**:
```
┌─────────────────────────────────────────────────────────┐
│ AI Portal                                    User-E10001│
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  WebSDK 加载中...                               │   │
│  │                                                 │   │
│  │  [iframe: sdk-host.html]                        │   │
│  │                                                 │   │
│  │  内部加载 embedLite.js                          │   │
│  │  初始化 HiagentWebSDK.WebLiteClient            │   │
│  │                                                 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**嵌入配置**:
```json
{
    "script_url": "http://127.0.0.1:4096/resources/product/llm/public/sdk/embedLite.js",
    "app_key": "kb_policy_key",
    "base_url": "http://127.0.0.1:4096/kb/chat",
    "launch_token": "xxx",
    "user_context": {
        "emp_no": "E10001",
        "name": "User-E10001",
        "dept": "demo",
        "email": "E10001@company.com"
    }
}
```

**状态**: ✅ 正常 - 嵌入配置获取成功，sdk-host.html 可访问

| | SDK 初始化 | ⚠️ 依赖第三方 SDK |

### 7.2 浏览器兼容性说明

- **Chrome/Edge**: ✅ 完全支持
- **Firefox**: ✅ 完全支持  
- **Safari**: ✅ 完全支持
- **移动端**: ⚠️ 需要响应式测试

### 7.3 已知限制

1. **WebSDK 脚本加载**: 依赖 OpenCode 服务器的 `embedLite.js`，如果该脚本不存在会显示加载失败
2. **跨域问题**: iframe 和 parent 之间通过 postMessage 通信，需要确保域名配置正确
3. **移动端适配**: 当前主要优化了桌面端体验，移动端需要进一步测试

### 7.4 建议

1. 添加前端错误监控（如 Sentry）
2. 添加加载性能监控
3. 优化移动端响应式布局
4. 添加前端缓存策略
